Installation¶

Warning

Sage is currently alpha-stage software. The API will be changing often. Please take this into consideration before you begin any significant development with Sage.

Sage works with CPython version 2.7 and greater, but it will not work (yet) with Python 3 as Twisted (a dependency) hasn’t fully supported it yet. Sage also can be used with PyPy.

It’s also recommended to have libyaml installed but not required.

Using Pip¶

Pip makes installing Sage and its dependencies painless.

pip install -e git+https://github.com/astralinae/sage.git#egg=sage
