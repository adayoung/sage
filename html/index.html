Welcome to Sage¶

Welcome to Sage’s documentation. This documentation is divided into different parts. I recommend you get started with Installation and then head over to the Quickstart. After the quickstart, there are more detailed examples that will show you how to create complete (albeit small) applications with Sage. If you’d rather dive into the internals of Sage, check out the API documentation.
